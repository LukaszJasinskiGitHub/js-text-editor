# Ten plik README zawiera informacje dotyczące uruchamiania JS edytora tekstu.

Aby lokalnie uruchomić aplikację należy:

```
 npm install
```

oraz:

```
npm start
```

następnie przejdź do aplikacji: http://localhost:3002
